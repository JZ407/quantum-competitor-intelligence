## 产品 (5款)
- **Aquila** (量子计算机, 商用)
  256-qubit 中性原子量子计算机。基于铷原子光镊阵列，支持模拟和数字模式。已在 AWS Braket 上提供云服务。
  发布: 2022-11
- **Gemini-Class 系统** (量子计算机 (研发中), 研发中)
  下一代系统，支持横向逻辑门(Transversal Gates)，纠错能力提升。目标：1000+逻辑量子比特。
  发布: 2028-2029
- **Bloqade** (软件SDK, 商用)
  中性原子量子计算开源编程框架。支持模拟和数字模式，脉冲级控制。Python SDK。
  发布: 2023
- **Tsim** (模拟器工具, 商用)
  快速通用量子错误校正模拟器。支持非克利福德电路模拟，用于验证纠错方案性能。
  发布: 2026-04
- **Amazon Braket 集成** (云服务, 商用)
  Aquila 是首个上架 AWS Braket 的中性原子量子计算机。通过 qBraid 平台也可访问。
  发布: 2023

- 顶尖学术血统: Lukin(h-index 176, 美国科学院院士)+3位哈佛/MIT教授联合创办，346位论文作者中Top15有12位直接关联

- 云平台独占: Aquila 是 AWS Braket 上唯一的中性原子量子计算机，渠道壁垒高
- 全栈能力: Aquila硬件+ Bloqade SDK + Tsim纠错模拟器 + 33家合作伙伴，覆盖量子计算全价值链
- 纠错路线图明确: Transversal STAR 架构已获专利保护，目标2028年1000+逻辑比特，99.9999999%保真度
- 全球化布局: 波士顿总部+日本AIST(1M合同)+英国Harwell+澳洲Pawsey，三大洲运营
- 董事会含金量高: Paul Maritz(VMware前CEO/Microsoft早期高管)+Arthur Chu+Google/软银代表
- 论文-产品闭环: 89篇论文直接支撑5款产品，学术产出与商业交付高度同步

### 劣势 (7项)
- 硬件复杂度高: 中性原子需要超低温真空+精密激光系统，工程成本大、可靠性挑战高
- 数字门模式待提升: Aquila 当前以模拟模式为主，数字门保真度仍落后于超导和离子阱路线
- 品牌认知度不足: 相比 IBM/Google 等巨头，QuEra 在非量子专业市场的知名度有限
- 核心人物依赖: 技术路线高度绑定 Lukin/Vuletić/Greiner/Englund 四位学术创始人
- 营收规模小: 虽有30M融资但商业化收入仍处早期，1M AIST合同是最大单项收入
- 员工规模有限: ~150人对标00M+估值，工程交付和客户支持能力存疑
- 物理比特数落后: 256 qubit 已被部分竞争者超越，需依赖纠错实现"逻辑比特"差异化

### 机会 (8项)
- FTQC先发优势: 若2028年成功交付1000+逻辑比特，将成为首个实用化容错量子计算平台
- AWS Braket 渠道放量: 作为唯一中性原子供应商，享受 AWS 全球企业客户流量
- 亚太市场: 日本AIST合同是桥头堡，可辐射韩国、新加坡、澳大利亚量子计算需求
- HPC融合大趋势: Pawsey/HPE/NERSC 等超算中心争相部署量子加速器，QuEra已占先机
- Google深度合作潜力: 从投资方升级为技术合作伙伴(TensorFlow Quantum集成、TPU协同)
- IPO窗口: 量子计算赛道若在2027-2028年整体升温，QuEra是候选IPO标的
- 常温运行优势: 铷原子无需稀释制冷机，相比超导路线的mK级制冷能耗优势显著
- 合作伙伴网络效应: 33家伙伴覆盖云/软件/咨询/HPC，形成生态系统护城河
